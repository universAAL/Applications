Error: "No AALSpace default configuration found on the path: configurations/etc/„

solution

Set absolute path for 
aalSpaceConfigurationPath=configurations/etc
aalSpaceSchemaURL=configurations/etc
E.g.: 
aalSpaceConfigurationPath=C:/universAAL/Training_Context/rundir/configurations/etc
aalSpaceSchemaURL=C:/universAAL/Training_Context/rundir/configurations/etc
