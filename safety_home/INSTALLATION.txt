
INTSTALLATION
-------------

1. Copy and paste the utils\sounds folder
   in rundir\confadmin\safety folder.

2. Import each Eclipse project.

3. Execute "Maven Install" for 
   each project.

4. Import the utils/safety_UI_mw1.3.0.launch
   launch file in the utils folder. 

5. Copy and paste the utils\ui.dm folder
   in rundir\confadmin\
